applicaton.properties:
bot.name - имя вашего телеграм-бота;
bot.token - токен вашего телеграм-бота, который используется для аутентификации бота в Telegram API;
bot.text.model - модель языковой генерации, используемая для генерации текста с помощью OpenAI API;
bot.text.temperature - параметр температуры модели языковой генерации, который определяет степень случайности
генерируемого текста. Значения между 0 и 1 уменьшают случайность, а значения больше 1 увеличивают ее;
bot.text.max-tokens - максимальное количество токенов (слов), которые могут быть сгенерированы моделью языковой
генерации;
bot.text.top-p - параметр top-p, который используется для сокращения числа сгенерированных слов в результате.
Он указывает, какой процент наиболее вероятных вариантов ответов следует рассматривать. Значения между 0 и
1 уменьшают число возможных ответов, а значения больше 1 увеличивают его;
bot.text.frequency-penalty - штраф за повторение слов в ответе. Он указывает, насколько сильно модель должна избегать
повторения слов в своих ответах. Значения между 0 и 1 уменьшают штраф, а значения больше 1 увеличивают его;
bot.text.presence-penalty - штраф за неприемлемые слова в ответе. Он указывает, насколько сильно модель должна избегать
использования неприемлемых слов в своих ответах. Значения между 0 и 1 уменьшают штраф, а значения больше 1 увеличивают
его;
api.token - токен OpenAI API, который используется для аутентификации в OpenAI API;
api.url.completions - URL-адрес API-запроса для генерации текста с помощью OpenAI API.

OpenAI API:
Для того, чтобы получить токен OpenAI API, необходимо создать учетную запись на сайте OpenAI и затем создать ключ API.
Для этого необходимо выполнить следующие шаги:

Зарегистрируйтесь на сайте OpenAI, перейдя по ссылке https://beta.openai.com/signup/.
Подтвердите свой адрес электронной почты, следуя инструкциям, которые вы получите по электронной почте после
регистрации.
Войдите в свою учетную запись на сайте OpenAI, перейдя по ссылке https://beta.openai.com/login/.
Перейдите на страницу API Keys, перейдя по ссылке https://beta.openai.com/docs/api-reference/authentication.
Нажмите кнопку "Create API Key", чтобы создать новый ключ API.
Скопируйте токен, который будет сгенерирован, и сохраните его в безопасном месте.
Теперь у вас есть токен OpenAI API, который вы можете использовать для аутентификации в OpenAI API.
Не забудьте сохранить его в файле настройки вашего приложения, чтобы ваш Java Spring Boot приложение могло его
использовать для общения с OpenAI API.

Telegram API:
Для того, чтобы получить токен Telegram API, необходимо создать телеграм-бота. Для этого необходимо выполнить
следующие шаги:
1. Откройте Telegram и найдите @BotFather.
2. Нажмите кнопку "Start".
3. Нажмите кнопку "Send a message to @BotFather".
4. Напишите команду "/newbot" и нажмите кнопку "Send".
5. Напишите имя вашего бота и нажмите кнопку "Send".
6. Напишите имя пользователя вашего бота и нажмите кнопку "Send".
7. Вы получите сообщение с токеном вашего бота. Скопируйте его и сохраните в безопасном месте.
8. Теперь у вас есть токен Telegram API, который вы можете использовать для аутентификации в Telegram API.
Не забудьте сохранить его в файле настройки вашего приложения, чтобы ваш Java Spring Boot приложение могло его
использовать для общения с Telegram API.

Создание модели языковой генерации:
Для того, чтобы создать модель языковой генерации, необходимо выполнить следующие шаги:
1. Откройте сайт OpenAI и войдите в свою учетную запись.
2. Перейдите на страницу API Keys, перейдя по ссылке https://beta.openai.com/docs/api-reference/authentication.
3. Нажмите кнопку "Create API Key", чтобы создать новый ключ API.
4. Скопируйте токен, который будет сгенерирован, и сохраните его в безопасном месте.
5. Перейдите на страницу Engines, перейдя по ссылке https://beta.openai.com/docs/api-reference/engines.
6. Нажмите кнопку "Create Engine", чтобы создать новый двигатель.
7. Введите название двигателя и нажмите кнопку "Create Engine".
8. Вы получите сообщение с идентификатором двигателя. Скопируйте его и сохраните в безопасном месте.
9. Теперь у вас есть идентификатор двигателя, который вы можете использовать для создания модели языковой
генерации.
10. Перейдите на страницу Engines, перейдя по ссылке https://beta.openai.com/docs/api-reference/engines.
11. Нажмите кнопку "Create Engine", чтобы создать новый двигатель.
