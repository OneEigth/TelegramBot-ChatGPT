Этот код описывает класс MyBot, который является телеграм-ботом и использует Spring Framework для управления
зависимостями и инжектирования сервисов.

Аннотация @Component указывает Spring, что этот класс является компонентом и может быть использован в качестве бина.

Класс MyBot расширяет класс TelegramLongPollingBot, который предоставляет методы для подключения к Telegram API.

В методе onUpdateReceived бот принимает сообщение от пользователя и выполняет два метода pong и askGpt,
которые описывают ответы бота на сообщения пользователя.

Метод askGpt использует сервис ChatGPTService для обработки сообщения и отправляет ответ пользователю
с помощью метода execute.

Метод pong отправляет сообщение пользователю, подтверждающее получение его сообщения.

Методы getBotUsername и getBotToken возвращают имя и токен бота, соответственно.

Методы setService и setEnv используют аннотацию @Autowired, чтобы Spring Framework мог инжектировать
ChatGPTService и Environment в этот класс.
